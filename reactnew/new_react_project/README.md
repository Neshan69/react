# React Project Create Step
## step by step procedure

```js
//Creating react project with vite
npm create vite@latest
```
### Steps:
- Set project name 
- Select framework: `React`
- select Varient (programming language): `***Javascript***`
- Install npm and start? Yes
    - If you type `yes`
        - It install and provide you the localhost url
        - Opn the localhost url on browser
    - If `No`
        - cd [Project name]
        - `npm install`
        - `npm run dev`
